# MTIP Collective

Landing site for MTIP Collective — collective.mtip.ai

## Pages

- `/` → root (redirects to `/pods`)
- `/pods` → MTIP Collective Pods (full product page: The Studio, The Duo, The Quad, The Suite)

## Assets

- `pods/assets/video/walkthrough.mp4` — hero walk-through video
- `pods/assets/brochure/mtip-collective-pods-brochure.pdf` — full PDF (manufacturer brochure, awaiting MTIP rebrand)
- `pods/assets/images/` — favicon, OG image, pod renders

## Stack

Single-file static HTML. No build step. Deployable via Lovable, Vercel, Netlify, GitHub Pages, or any static host.

## Brand

- Powered by Loren AI (footer credit, mandatory)

- Typography: Cormorant Garamond (display) + Manrope (body)
